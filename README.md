# RISC-V RV32I Processor Functional Verification

SystemVerilog | UVM | QuestaSim

Team Project (3 Members)

---

## Project Overview

This project focuses on functional verification of a 32-bit Single-Cycle RISC-V RV32I processor.

The verification environment was developed using UVM methodology to validate processor functionality through coverage-driven and assertion-based verification.
